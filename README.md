# Pineapple-Studio

A full-fledged IDE for a Minecraft mod called ComputerCraft.

### About the project

Pineapple-Studio was created with the incentive to make in-game editing actually enjoyable. WIP.

### Notable (planned) features
- Themes
- Extensions
- Project contexts
- Custom interpreter and pre-build script
- Searching within files, project and actions
- Multiple open files
- Auto-saving
- Syntax highlighting
